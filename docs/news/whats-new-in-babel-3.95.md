# What's new in babel 3.95

2023-09-29

There are no changes in the core code.

## New locales

There are 13 new locales:

* Cebuano
* Chuvash
* Dogri
* Northern Frisian
* Hmong Njua
* Maori
* Manipuri
* Santali
* Sindhi
* Sundanese
* Tajik
* Tatar
* Xhosa    

In addition, there are 3 bare minimum locales:
* Awadhi
* Gothic
* Saraiki.

A few missing native numerals and character ranges has been added.
